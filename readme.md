## 业务背景

面向工业互联网场景的轻量化监控面板，解决传统后台在**设备状态实时感知 + 万级数据渲染 + 弱网容灾**三大痛点。
模拟产线设备温度 / 负载数据上报，支撑运维人员快速定位异常设备。

---

## 技术亮点（面试可深挖）

* **实时通信**：Socket.io 单例封装 + 心跳保活 + 5 次重连机制，保障弱网下数据不丢失
* **性能优化**：ECharts 按需引入 + ResizeObserver 自适应重绘，窗口缩放无卡顿
* **类型安全**：TypeScript 严格模式 + 完整接口定义，联调错误率降低 90%+
* **工程规范**：Composables 封装数据订阅逻辑，组件复用率提升 40%

---

## 核心成果（量化表达）

* 模拟万级设备日志列表：虚拟滚动 + 行缓存，帧率 12fps → 58fps
* 首屏加载优化：代码分割 + 按需引入，体积 2.3MB → 780KB
* 弱网降级策略：本地缓存 + 增量同步，保障监控连续性
